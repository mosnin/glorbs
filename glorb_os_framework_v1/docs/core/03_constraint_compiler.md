# 03 Constraint Compiler

## Constraint Inputs

1. Deadline
2. Budget
3. Runtime limits
4. Quality bar
5. Risk tolerance
6. Reversibility
7. Legal or policy boundaries
8. Tool availability
9. Human approval requirements

## Compiler Outputs

Constraints should influence:

1. Topology choice
2. Agent count
3. Critique intensity
4. Memory policy
5. Deliverable depth
6. Runtime selection
7. Human oversight requirements
