# 07 Failure Recovery

## Failure Classes

1. Invalid mission definition
2. Poor routing
3. Role overlap
4. Tool failure
5. Runtime mismatch
6. Memory conflict
7. Failed merge
8. Quality gate failure
9. Policy violation

## Recovery Strategies

1. Retry
2. Recompile topology
3. Spawn a specialist
4. Escalate to human review
5. Reduce scope
6. Switch runtime backend
7. Rebuild deliverable from preserved provenance
